# 课程体系设计

## 1. 概述

本文档设计了一套适合6岁儿童的积木编程课程体系，通过游戏化、渐进式的学习路径，培养儿童的逻辑思维和问题解决能力。

---

## 2. 课程设计原则

### 2.1 认知发展阶段
6岁儿童的认知特点：
- **具体形象思维为主**：需要直观的视觉和操作体验
- **注意力持续时间短**：15-20分钟专注，需要游戏化互动
- **即时反馈需求**：每个操作都需要明确的视觉反馈
- **成就感驱动**：通过小目标建立自信

### 2.2 教学方法
- **游戏化学习**：将知识点融入游戏场景
- **试错学习**：鼓励实验，不怕失败
- **脚手架支持**：从示范到模仿再到创造
- **多感官参与**：视觉、听觉、触觉综合体验

## 4. 入门阶段课程

### 课程1: 小车第一次动起来

**时长**: 15分钟
**目标**: 认识积木块，学会基础运动控制

**知识点**:
- 认识积木工具箱
- 拖拽积木到工作区
- 连接积木块
- 运行代码

**积木块**:
- 前进
- 后退
- 停止

**任务流程**:
```
1. 找到"运动"工具箱 🎯
2. 拖一个"前进"积木 📦
3. 拖一个"等待2秒"积木 ⏱️
4. 拖一个"后退"积木 🔄
5. 拖一个"停止"积木 🛑
6. 点"运行"按钮 ▶️
```

**预期效果**: 小车前进2秒后后退

**趣味包装**: "帮小车走出第一步！"

---

### 课程2: 会跳舞的小车

**时长**: 15分钟
**目标**: 学习方向控制，组合多个动作

**知识点**:
- 左右移动
- 旋转动作
- 动作序列

**积木块**:
- 前进/后退
- 左平移/右平移
- 左转/右转
- 停止
- 等待

**任务流程**:
```
1. 前进2秒 ⬆️
2. 等待1秒 ⏸️
3. 后退2秒 ⬇️
4. 等待1秒 ⏸️
5. 左转1圈 🔄
6. 停止 🛑
```

**预期效果**: 小车完成一个简单的舞蹈动作

**趣味包装**: "让小车跳一支舞！"

---

### 课程3: 云台摄影师

**时长**: 15分钟
**目标**: 学习云台控制

**知识点**:
- 云台上下控制
- 云台左右控制
- 云台复位

**积木块**:
- 云台向上/下
- 云台向左/右
- 云台复位

**任务流程**:
```
1. 云台向上30度 👆
2. 等待2秒
3. 云台向下30度 👇
4. 等待2秒
5. 云台向左30度 👈
6. 等待2秒
7. 云台向右30度 👉
8. 等待2秒
9. 云台复位 🔄
```

**预期效果**: 摄像头云台上下左右转动

**趣味包装**: "当一次小摄影师！"

---

### 课程4: 我的第一盏灯

**时长**: 15分钟
**目标**: 学习输出控制

**知识点**:
- LED灯控制
- 蜂鸣器控制
- 视觉反馈

**积木块**:
- LED灯
- 蜂鸣器

**任务流程**:
```
1. LED灯设为红色 🔴
2. 等待1秒
3. 蜂鸣器开启 🔊
4. 等待1秒
5. LED灯设为绿色 🟢
6. 等待1秒
7. 蜂鸣器关闭 🔇
8. LED灯关闭 ⚫
```

**预期效果**: LED闪烁并伴随蜂鸣

**趣味包装**: "做一个小信号灯！"

## 5. 进阶阶段课程

### 课程5: 会看路的小车

**时长**: 20分钟
**目标**: 学习超声波传感器和条件判断

**知识点**:
- 超声波传感器
- 条件判断（如果...就...）
- 比较大小

**积木块**:
- 超声波距离
- 如果...就...
- 比较积木（小于/大于）
- 前进/停止

**任务流程**:
```
重复执行:
    距离 = 获取超声波距离
    如果 距离 < 200mm:
        停止 🛑
        蜂鸣器响1秒 🔊
    否则:
        前进(慢速) ⬆️
```

**预期效果**: 小车检测到障碍物自动停止

**趣味包装**: "给小车装上眼睛！"

---

### 课程6: 追颜色的小狗

**时长**: 25分钟
**目标**: 学习颜色识别和循环

**知识点**:
- 颜色识别
- 循环结构
- 视觉反馈

**积木块**:
- 检测颜色
- 寻找颜色
- 如果...就...否则...
- 追踪颜色
- LED灯

**任务流程**:
```
重复执行:
    颜色 = 检测颜色
    如果 颜色 == 红色:
        LED灯设为红色 🔴
        停止 🛑
    否则:
        LED灯设为绿色 🟢
        继续追踪红色 🎯
```

**预期效果**: 小车自动追踪红色物体

**趣味包装**: "训练一只追颜色的小狗！"

---

### 课程7: 巡线小侦探

**时长**: 25分钟
**目标**: 学习巡线传感器

**知识点**:
- 巡线传感器原理
- 根据传感器状态判断
- 复杂条件判断

**积木块**:
- 巡线状态
- 如果...就...否则...
- 左平移/右平移
- 前进

**任务流程**:
```
重复执行:
    状态 = 获取巡线状态
    如果 状态[1] 和 状态[2] 都是 True:
        前进(中速) ⬆️
    否则如果 状态[0] 是 True:
        左转(慢速) ↪️
    否则如果 状态[3] 是 True:
        右转(慢速) ↩️
    否则:
        停止 🛑
```

**预期效果**: 小车沿黑线行驶

**趣味包装**: "当一次小侦探，沿着线索走！"

---

### 课程8: 智能避障车

**时长**: 30分钟
**目标**: 综合运用传感器和逻辑

**知识点**:
- 多传感器配合
- 复杂逻辑判断
- 优化算法

**积木块**:
- 所有已学积木

**任务流程**:
```
重复执行:
    距离 = 获取超声波距离
    如果 距离 < 300mm:
        # 有障碍物，转向避开
        左转(慢速) 1秒 ↪️
        检查新的距离
        如果 新距离 > 300mm:
            前进(中速) 2秒 ⬆️
        否则:
            右转(慢速) 1秒 ↩️
    否则:
        前进(快速) ⬆️
```

**预期效果**: 小车自动避开障碍物

**趣味包装**: "造一辆聪明的避障车！"

## 6. 创作阶段课程

### 课程9: 我的小车舞步

**时长**: 开放
**目标**: 自由创作，表达创意

**开放任务**:
- 设计一套小车舞蹈动作
- 结合灯光和声音
- 可以表演给家人看

**示例参考**:
```

LED闪烁配合 → 蜂鸣器节奏
```

## 8. 家长指导

### 8.1 陪伴建议

1. **不急于帮忙**：让孩子自己尝试，失败也是学习
2. **多鼓励提问**：引导孩子思考"为什么"
3. **记录进步**：拍照或录像记录孩子的作品
4. **分享成果**：让孩子向家人展示作品

### 8.2 常见问题

| 问题 | 建议 |
|------|------|
| 孩子不会怎么办 | 从示范开始，让孩子模仿 |
| 孩子失去兴趣 | 切换到更有趣的任务 |
| 程序出错 | 引导孩子检查哪里不一样 |
| 想要更多挑战 | 尝试创作阶段课程 |

## 9. 评估标准

### 9.1 技能评估

| 阶段 | 技能要求 |
|------|----------|
| 入门 | 能拖拽积木、连接积木、运行程序 |
| 进阶 | 能使用传感器、条件判断、循环 |
| 创作 | 能独立设计程序、解决复杂问题 |

### 9.2 兴趣评估

- 是否主动要求学习
- 是否愿意尝试新任务
- 是否有创造性想法
- 是否能专注完成任务
